✦ Instagram Feature Concept

Reels Queue
in DMs

When friends send multiple reels, you currently bounce back to chat after every single one. This feature makes them flow into a seamless, scrollable queue.

😩

Today

Watch reel → go back → tap next → go back → repeat. 3 reels = 6+ taps.

✨

With Queue

Tap once, watch all consecutive reels in one session. Stops at the next text message.
